# IPL_2025
# 🏏 IPL 2025 Performance Analytics Dashboard

## 📌 Project Overview

This project analyzes IPL 2025 data using Python, SQL, MySQL, and Power BI to generate meaningful insights about team performance, batting statistics, bowling statistics, and match trends.

The project follows a complete Data Analytics workflow, including data preparation, exploratory data analysis (EDA), SQL-based analysis, database integration, and dashboard development.

---

## 🎯 Objectives

- Analyze IPL 2025 team and player performance
- Identify top batsmen and bowlers
- Evaluate venue-wise match trends
- Generate actionable cricket insights
- Build an interactive Power BI dashboard

## 🔄 Project Workflow

### 1. Data Preparation
- Imported IPL datasets
- Checked data quality
- Handled missing values
- Prepared data for analysis

### 2. Exploratory Data Analysis (EDA)
- Team performance analysis
- Venue analysis
- Run distribution analysis
- Wicket distribution analysis
- Player performance analysis

### 3. SQL Analysis
Performed analytical SQL queries such as:

- Team with most wins
- Toss impact analysis
- Top run scorers
- Top wicket takers
- Most fours and sixes
- Player of the Match leaders
- Team-wise runs analysis
- Team-wise wickets analysis

### 4. MySQL Integration
- Created IPL database
- Imported CSV datasets into MySQL
- Executed analytical SQL queries

### 5. Power BI Dashboard Development
Created a multi-page interactive dashboard.
